Реализованные механики:
  1.Уворот от атак - периодически враг пытается атаковать. В этот момент на экране появляется плашка 
  с указанием слабого места для увертывания. Нажмите стрелочку на клавиатуре в соответствующем 
  направлении, чтобы увернуться от атаки.
  2.Зелья - с убитых врагов выпадают зелья. Их можно сохранять и использовать в любое время. Можно 
  использовать несколько зелий одновременно, эффект будет пропорционально увеличиваться.

Сохранение в игре:
  Игра сохраняется при выходе через главное меню и автоматически загружается, если есть сохранение. 
  Сохраняется объект Game, который представляет всю игру. Для этого достаточно перевести его в 
  формат JSON, чтобы иметь возможность получить всю информацию о состоянии игры в дальнейшем. Таким 
  образом, JSON от Game является своего рода снимком в конкретный момент времени.

Управление:
  Клики мышью по врагу - атаковать его; по зельям - использовать их. Стрелочки - увороты.
